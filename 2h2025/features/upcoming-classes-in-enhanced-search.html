Upcoming Classes in Enhanced Search

Feature #7 of 32
End User

What Is This Feature?

Learners can now see upcoming class schedules directly within enhanced search results. When a search matches a course with available upcoming classes, a Show Classes option expands a list of up to five classes sorted by date, showing Date & Time, Timezone, and Facility. The list updates dynamically based on applied filters. Learners can select a class to view Class Details or navigate to the full Available Classes page. This feature replaces the calendar view from the legacy Learning experience and was requested via Customer Community Idea 325775.

What Is Changing?

A new Show Classes toggle appears in enhanced search results for courses with upcoming classes, displaying up to five classes with schedule details. This replaces the legacy calendar view.

Why Should We Care?

Learners can compare class schedules, locations, and availability directly from search results, reducing the clicks needed to find and register for the right class.

Risk of Inaction

No action needed — automatically enabled. Requires enhanced search to be enabled.
