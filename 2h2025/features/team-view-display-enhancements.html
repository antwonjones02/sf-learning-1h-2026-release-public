Team View Display Enhancements

Feature #18 of 32
Manager

What Is This Feature?

Several display improvements have been made to the redesigned Team View page: the default number of assignment cards loaded per view has increased from 8 to 20; Organization ID and User ID values are now shown on the user detail page for easier identification; and team hierarchy navigation has been updated with a persistent Back button (always moves up one level) and a new persistent Return to Direct Reports option. Based on Customer Influence Idea 336808.

What Is Changing?

Team View now loads 20 assignment cards by default (up from 8), shows Organization ID and User ID on user detail pages, and provides improved persistent navigation controls for team hierarchy traversal.

Why Should We Care?

Managers working with larger teams can view more assignments at once and navigate the team hierarchy more efficiently, reducing the number of clicks required to manage their team's learning.
