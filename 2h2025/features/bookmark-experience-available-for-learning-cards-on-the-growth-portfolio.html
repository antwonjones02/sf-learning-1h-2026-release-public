Bookmark Experience Available for Learning Cards on the Growth Portfolio

Feature #9 of 32
End User

What Is This Feature?

Users can now bookmark Learning items directly from the Learning cards displayed in Growth Portfolio, using the same bookmark icon available in the Learning application. Previously, the bookmark icon was not available on Learning items within Growth Portfolio.

What Is Changing?

A bookmark icon is now shown on Learning cards within Growth Portfolio, providing a consistent bookmarking experience across Growth Portfolio and Learning.

Why Should We Care?

Users get a seamless, consistent experience between Growth Portfolio and Learning, making it easier to save and revisit learning content directly from their development workspace.

Risk of Inaction

No action needed — automatically enabled.

Recommendation No action required
Type Changed
Impact Minor
