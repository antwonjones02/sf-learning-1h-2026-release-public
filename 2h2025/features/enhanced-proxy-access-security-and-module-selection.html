Enhanced Proxy Access Security and Module Selection

Feature #4 of 32
AdminEnd User

What Is This Feature?

SAP has improved Proxy access security and expanded module selection options for SAP SuccessFactors Learning. Proxy users can now only access features and data explicitly granted to them, reducing the risk of unauthorized access. Admins manage proxy modules via Admin Center > Proxy Management > Grant Proxy Rights, and users can assign proxy access in Settings > Proxy > Assign Proxy.

What Is Changing?

The list of proxy modules and sub-modules has been updated for 2H 2025, tightening which areas a proxy user can access. Customers should review the updated proxy module list against their existing proxy configurations.

Why Should We Care?

Tighter proxy access controls reduce security risk by ensuring proxy users cannot access data or features beyond what is explicitly granted, supporting compliance and data governance.

Risk of Inaction

No action needed — automatically enabled. However, SAP recommends reviewing proxy configurations to ensure they align with the updated module list.
