Updated Learning Cross-Domain Proxlet

Feature #7 of 32
Admin

What Is This Feature?

The cross-domain proxlet for SAP SuccessFactors Learning customers who host their own content has been updated to align with the Learning platform's upgrade to OpenJDK (Open Java Development Kit) version 21. Customers must first upgrade OpenJDK to version 21 on their content-hosting server, then download the latest 2025 cross-domain proxlet from Support Downloads.

What Is Changing?

The cross-domain proxlet has been updated to support OpenJDK 21. Customers using self-hosted content must upgrade both OpenJDK and the proxlet to continue operating correctly.

Why Should We Care?

Using the latest proxlet ensures compatibility with the platform's updated Java runtime, preventing potential content delivery issues for self-hosted learning content.

Risk of Inaction

Customers who do not update to the 2025 proxlet after the OpenJDK 21 upgrade may experience issues delivering self-hosted learning content. SAP recommends using the latest version for best results.
