Universal Rollout of the New Learning Home Page

Feature #6 of 32
End UserAdmin

What Is This Feature?

The new Learning home page is now universally enabled for all customers. The toggle that allowed learners to switch between the legacy and new experiences has been removed, as has the related default setting on the Integrated Learning Settings page. Validated Learning (vSaaS) customers are not impacted in this release and may continue using the toggle.

What Is Changing?

The New Learning Experience switch is removed for all non-vSaaS customers. Learners can no longer revert to the legacy home page. The default setting on Integrated Learning Settings is also removed.

Why Should We Care?

All customers now benefit from the modern Learning home page experience, standardizing the learner interface and enabling SAP to focus development effort on the new experience.

Risk of Inaction

No action needed — automatically enabled.
