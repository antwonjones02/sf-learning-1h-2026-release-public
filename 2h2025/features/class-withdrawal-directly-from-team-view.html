Class Withdrawal Directly from Team View

Feature #2 of 32
ManagerEnd User

What Is This Feature?

Managers can now withdraw learners directly from classes using Team View, either via the More Actions menu on the Team View page or the Withdraw button on the Class Details page. This replaces the previous requirement to use the bulk withdrawal tool for all manager-initiated withdrawals. Delivered in response to Customer Influence Idea 334035.

What Is Changing?

A new Withdraw option appears in the More Actions menu in Team View. Previously, managers could only withdraw subordinates through the bulk withdrawal tool.

Why Should We Care?

Managers can act quickly on class cancellations or scheduling conflicts directly from their Team View dashboard, without navigating to separate admin tools.

Risk of Inaction
