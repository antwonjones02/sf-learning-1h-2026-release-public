Enhancements to Manager View in Latest Career Worksheet

Feature #8 of 32
ManagerAdmin

What Is This Feature?

Three enhancements have been made to the manager view (Summary tab) in the latest Career Worksheet within Growth Portfolio: private-access permission is now supported so managers only see target roles when explicitly permitted; managers can now view employees' unrated skills and attributes with an Unrated tag; and managers can access linked development activities (goals, assignments, learning) for specific skills via a Development Activities link. These changes achieve feature parity with the legacy Career Worksheet.

What Is Changing?

Managers gain visibility into employees' unrated skills and development activities linked to skills. Target role visibility is now gated by private-access permission, which was not enforced previously.

Why Should We Care?

Managers receive a more complete picture of employee skill readiness and development progress, enabling better coaching conversations and career planning support.

Risk of Inaction

No action needed — automatically enabled. Admins should review Career Worksheet template permission settings to ensure private-access is configured as intended.
