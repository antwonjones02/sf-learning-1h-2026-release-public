Deprecation of SuccessFactors Learning Token Server

Feature #4 of 32
Admin

What Is This Feature?

The SuccessFactors Learning token server is deprecated and will be deleted on November 20, 2026. The Learning token server uses HTTP basic authentication (Client ID and Secret), which is less secure than the platform token server's support for SAML Assertion Based token requests. Customers should migrate integrations to OAuth 2.0 or OpenID Connect authentication using the platform token server.

What Is Changing?

The Learning token server reaches its Deleted milestone on November 20, 2026 and will no longer be available for API authentication. Integrations must be migrated to the platform token server using OAuth 2.0 or OpenID Connect.

Why Should We Care?

The platform token server provides more secure authentication mechanisms (SAML Assertion Based) compared to the legacy Learning token server, reducing security risk for API integrations.

Risk of Inaction

Integrations using the Learning token server for authentication will stop working after November 20, 2026. Organizations must plan and execute migration to OAuth 2.0 or OpenID Connect before the deletion date.

Recommendation Migrate to OAuth NOW
