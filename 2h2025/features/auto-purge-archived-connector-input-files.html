Auto-Purge Archived Connector Input Files

Feature #3 of 32
Admin

What Is This Feature?

Archived connector input files are now automatically purged after 30 days by default, eliminating the need for manual retention configuration. When the Enable input file archive option is activated, the previously configurable Purge archived input files older than setting is removed. SAP recommends downloading the zip file of archived input files before the 30-day window closes for troubleshooting purposes.

What Is Changing?

The configurable retention period setting for archived connector input files is removed and replaced with a fixed 30-day auto-purge. Admins no longer need to manually set the purge schedule.

Why Should We Care?

Automatic purging of archived connector files reduces manual admin overhead and keeps the system lean, preventing accumulation of stale files that can impact performance.

Risk of Inaction

Without enabling the archive feature, connector input files are not archived and therefore not subject to auto-purge. If archiving is in use, the fixed 30-day window replaces any previously configured retention period.
