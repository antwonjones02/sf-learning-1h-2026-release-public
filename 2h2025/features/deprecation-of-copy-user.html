Deprecation of Copy User

Feature #1 of 32
Admin

What Is This Feature?

The Copy User functionality in SAP SuccessFactors Learning has been deleted as of November 15, 2025. The Copy User menu item has been removed from Learning Administration > People > Users > Actions > Copy. This feature was an artifact of legacy Plateau products and did not function correctly in the current SAP SuccessFactors Learning environment.

What Is Changing?

The Copy User option is permanently removed from the Users page in Learning Administration. Customers who relied on this functionality must use alternative methods to replicate user configurations.

Why Should We Care?

Removing a non-functional legacy feature reduces confusion and clutter in the admin UI, and prompts customers to adopt correct, supported methods for user creation and configuration.

Risk of Inaction

This feature has already been deleted as of November 15, 2025. Any workflows or processes relying on Copy User must already be replaced with alternative approaches.
