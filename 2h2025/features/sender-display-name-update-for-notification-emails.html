Sender Display Name Update For Notification Emails

Feature #17 of 32
AdminManagerEnd User

What Is This Feature?

Admins can now configure display names for the From field in Learning notification emails. Three new fields have been added to Global Application Settings > Mail: Admin Email Display Name (for admin notifications), User Email Display Name (for user notifications), and Default Display Name (used when the other two fields are empty). Manager names now also appear next to their email address in manager-sent notifications such as overdue training reminders. Based on Customer Influence Idea 335406.

What Is Changing?

Three new display name fields are added to the Mail settings page in Global Application Settings. Manager names are now shown alongside email addresses in manager notification emails. Previously, no display name configuration existed.

Why Should We Care?

Branded, recognizable sender names improve email open rates and reduce the chance that notifications are marked as spam, ensuring learners and managers receive and act on important learning communications.

Risk of Inaction

No action needed — automatically enabled. Default behavior is unchanged unless admins configure the new display name fields.
