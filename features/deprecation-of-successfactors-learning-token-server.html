Deprecation of SuccessFactors Learning Token Server

Feature #51 of 34
Admin

What Is This Feature?

The Learning token server will reach end of maintenance on November 17, 2023 and will be deleted on November 20, 2026.

What Is Changing?

The legacy token-based authentication for Learning APIs is being permanently removed. Organizations must migrate to OAuth 2.0.

Why Should We Care?

After November 2026, any integrations using the legacy token server will break. This is a hard deadline.

Risk of Inaction

Critical: All integrations using the legacy token server will fail after November 20, 2026.

Recommendation Migrate to OAuth NOW
Type Deprecated
Impact Critical
