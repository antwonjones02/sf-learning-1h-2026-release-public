Horizon Visual Theme Opt-In Setting, Deleted

Feature #50 of 34
End UserAdmin

What Is This Feature?

The Horizon visual theme opt-in setting for Learning is deleted as of May 15, 2026. All Learning environments will automatically use the Horizon visual theme.

What Is Changing?

The ability to opt out of the Horizon theme is removed. Horizon becomes the mandatory visual standard.

Why Should We Care?

Standardizes the user experience across all Learning environments. Ensures consistency with the broader SAP design direction.

Risk of Inaction

No choice, this is automatic. Organizations still on the legacy theme will be migrated.

Recommendation Prepare for Horizon
