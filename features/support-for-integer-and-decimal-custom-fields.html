Support for Integer and Decimal Custom Fields

Feature #41 of 54
Admin

What Is This Feature?

Custom fields for the User entity now support Integer and Decimal data types.

What Is Changing?

Administrators can create numeric custom fields, previously limited to text and date types.

Why Should We Care?

Enables more precise user data tracking for scenarios like certification scores, experience levels, or custom metrics.

Risk of Inaction

Numeric data must be stored as text, limiting sorting, filtering, and reporting capabilities.

Recommendation Enable if needed
Type New
Impact Minor
