Reset E-Signature PIN on Learning Home Page

Feature #16 of 34
End User

What Is This Feature?

Learners can securely reset their e-signature PIN using a Forgot My PIN option directly from the Learning home page.

What Is Changing?

A self-service PIN reset flow is now accessible from the Learning home page, eliminating the need to contact an administrator.

Why Should We Care?

Reduces admin workload from PIN reset tickets. Unblocks learners who are stuck on compliance-critical e-signature activities.

Risk of Inaction

Learners who forget their PIN remain blocked until an admin manually resets it. Delays compliance completion.

Recommendation Enable
Type Changed
Impact Minor
