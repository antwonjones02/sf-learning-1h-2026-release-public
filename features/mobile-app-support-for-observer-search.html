Mobile App Support for Observer Search

Feature #27 of 34
End User

What Is This Feature?

Learners can now search for and add task observers directly in the SAP SuccessFactors Mobile iOS and Android apps.

What Is Changing?

Observer search (previously desktop-only) is now available on mobile devices.

Why Should We Care?

Unblocks mobile-first learners who need to complete task observations in the field.

Risk of Inaction

Mobile learners cannot assign observers, forcing them to use desktop for this step.

Recommendation Enable
Type New
Impact Minor
