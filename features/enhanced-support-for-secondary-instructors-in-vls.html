Enhanced Support for Secondary Instructors in VLS

Feature #30 of 54
Admin

What Is This Feature?

Virtual learning sessions now support enhanced management of secondary instructors alongside primary instructors.

What Is Changing?

Secondary instructors can be more effectively managed within virtual learning sessions, better visibility and control.

Why Should We Care?

Important for organizations running large virtual classes with co-facilitators. Improves session coordination.

Risk of Inaction

Secondary instructor management remains limited, requiring workarounds for co-facilitated sessions.

Recommendation Enable if using VLS
Type Changed
Impact Minor
