Class Withdrawal Directly from Team View

Feature #39 of 54
Manager

What Is This Feature?

Managers can now withdraw team members from classes directly from the Team View page.

What Is Changing?

Class withdrawal (previously a multi-step process) is now a single action from Team View.

Why Should We Care?

Speeds up class management and frees seats faster when team priorities change.

Risk of Inaction

Managers must navigate elsewhere or contact admin to withdraw team members from classes.

Recommendation Enable
Type New
Impact Minor
