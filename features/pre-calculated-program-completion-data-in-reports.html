Pre-Calculated Program Completion Data in Reports

Feature #5 of 34
AdminManager

What Is This Feature?

Administrators can now access pre-calculated program completion percentages in Data Extract Services. The legacy GetProgramCompletionPercentage operator is deprecated.

What Is Changing?

Program completion data is now pre-calculated and available for faster reporting. The legacy calculation method is being retired.

Why Should We Care?

Significantly faster and more reliable reporting on program completion. Eliminates performance issues with large program reports.

Risk of Inaction

No action needed, automatically enabled. Begin transitioning any reports that use the legacy operator.

Recommendation Update legacy reports
Type New
Impact Major
