Keyword Search on View All Pages

Feature #1 of 34
End User

What Is This Feature?

Learners can now search their learning activities (assignments, task observations, training requests, surveys, and bookmarks) by title or ID on the View All pages. Filtering by learning type is also supported.

What Is Changing?

A new search bar appears on all View All pages within the Learning home page, enabling quick lookups across all activity types.

Why Should We Care?

Reduces time learners spend scrolling through long lists to find specific assignments. Improves self-service and reduces help desk tickets.

Risk of Inaction

No action needed, this is automatically enabled. No risk.
